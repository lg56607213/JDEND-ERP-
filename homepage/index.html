핵심 기능

렌터카 업무 전체를 하나로

실제 렌터카 회사의 업무 흐름 그대로 설계했습니다. 필요한 기능만 골라 쓰세요.

🚗

차량 관리

발주 → 등록 → 배차 → 정비·검사 → 보험 → 매각까지 차량 한 대의 전 생애를 추적합니다.

  • 차량 발주·출고·등록 관리
  • 배차 현황과 정비·정기검사 이력
  • 보험 가입·만기 관리, 매각 처리
📝

계약 관리

고객·차량·계약을 연결해 계약 총괄부터 만기·해지까지 한 번에 관리합니다.

  • 계약 등록·조회·계약서 출력
  • 만기 임박 알림, 만기·중도 해지 처리
  • 법적 절차(미납 대응) 관리
💳

청구 · 수납

월 청구를 자동 생성하고 수납을 기록하면 미수·연체가 실시간으로 집계됩니다.

  • 청구 생성·세금계산서 발행
  • 현금·카드·계좌이체 수납 등록
  • 미수금·연체 현황 자동 집계
🧾

자동 회계 · 전표

수납·지출이 발생하면 전표가 자동 생성됩니다. 현금주의 기준으로 실무에 맞게.

  • 수납·지출 자동 전표 생성
  • 일계표·자금일보로 매일 확인
  • 계정과목·전표 승인 관리
📊

재무제표 · 감가상각

차량 감가상각과 재무제표를 자동 산출. 세무사에 넘길 자료를 버튼 한 번으로.

  • 재무상태표·손익계산서 자동
  • 차량별 감가상각 등록·조정
  • 여신·차입금 관리
🏢

멀티회사 관리

법인·사업장이 여러 개여도 한 계정에서 전환하며 관리합니다. 데이터는 회사별 분리.

  • 여러 회사/사업장 통합 관리
  • 회사별 사용자·권한 설정
  • 회사 간 데이터 완전 분리

왜 JDEND인가

범용 회계 프로그램이 아닌, 렌터카 전용

렌터카 업무 흐름을 그대로 담아 "따로 정리해서 입력"할 필요가 없습니다.

  • 보험 만기·정기검사 일정 관리

무료 상담·데모

2

자료 이전

보유 차량·계약·고객 자료를 옮겨 초기 세팅을 도와드립니다.

3

사용 교육

담당자 교육과 초기 안착을 지원합니다.

운영 시작

자주 묻는 질문

궁금한 점을 먼저 정리했습니다

지금 엑셀로 관리하던 자료를 옮길 수 있나요?

네. 도입 단계에서 보유 중인 차량·계약·고객 자료 이전을 지원합니다. 자세한 형식은 상담 때 안내드립니다.

회계를 잘 몰라도 쓸 수 있나요?

수납·지출만 입력하면 전표와 재무제표가 자동으로 만들어집니다. 회계 지식 없이도 세무사에 넘길 자료가 정리됩니다. (현금주의 기준)

회사(사업장)가 여러 개인데 각각 관리되나요?

한 계정에서 여러 회사를 전환하며 관리할 수 있고, 회사별 데이터는 서로 분리됩니다.

설치가 필요한가요?

웹 기반이라 별도 설치 없이 브라우저로 접속해 사용합니다. PC·노트북 어디서든 접속됩니다.

요금은 어떻게 되나요?

운영 차량 대수 기준의 월 구독제입니다. 자세한 금액은 요금제 페이지를 참고하세요. (현재 잠정 금액이며, 정식 출시 시 확정됩니다.)

먼저 화면으로 확인하세요. 무료입니다.
